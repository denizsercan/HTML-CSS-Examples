Marker Position

By default, the marker hands outside the content area for the list item, dis-
playing as a hanging indent. The list-style-position property allows you
to bull the bullet inside the content area so its run into the list content.

list-style-position

Values: inside | outsite | inherit

Default: outsite

Applies to: ul,ol and li (or elements whose display value is list-item)

inherits: yes

I've applied a background color to the list item in Figure 12-24 to reveal the
boundaries of their content area boxes. You can see that when the position
is set to outside(left), the markers fall outside the content area, and when it
is set to inside, the content area box extends to include marker.

li{background-color: #F99;}
ul#outside{ list-style-position: outside;}
ul#inside{list-style-position: inside;}

Make your Own bullets

You can also use your own image as a bullet using the list-style-image
property.

list-style-image

Values: url | none | inherit

Default: none

Applies to: ul,ol and li (or elements whose display value is list-item)

inherits: yes
